# 宝塔面板部署指南 (Baota Panel Deployment Guide)

本项目是一个纯静态网站（HTML/CSS/JS），非常适合使用宝塔面板进行快速部署。以下是详细步骤：

## 1. 环境准备
1. 登录你的宝塔面板。
2. 点击左侧 **软件商店**。
3. 搜索并安装 **Nginx**（任意版本，推荐 1.21 或 1.22，免费版即可）。
4. *本项目不需要 PHP、MySQL 或 Tomcat，选择“纯静态”即可。*

## 2. 添加站点
1. 点击左侧菜单的 **网站**。
2. 点击 **添加站点** 按钮。
3. 在弹出的窗口中填写：
   - **域名**：填写你的域名（如果没有域名，可以直接填写服务器的 **公网IP**）。
   - **根目录**：保持默认即可（通常为 `/www/wwwroot/你的域名或IP`）。
   - **FTP**：不创建。
   - **数据库**：不创建。
   - **PHP版本**：选择 **纯静态**。
4. 点击 **提交**。

## 3. 部署代码
这里提供两种方式，**推荐使用方式一**，方便后续一键更新。

### 方式一：使用终端 Git 拉取（最稳妥）
1. 点击面板左侧的 **终端**，输入服务器密码登录（如果是首次使用）。
2. 进入刚才创建的站点目录（请将 `your_domain_or_ip` 替换为你刚才填写的域名或IP）：
   ```bash
   cd /www/wwwroot/your_domain_or_ip
   ```
3. 删除默认生成的欢迎页面：
   ```bash
   rm -rf *
   ```
4. 克隆 GitHub 上的代码（**注意命令最后有一个空格和点**，表示克隆到当前目录）：
   ```bash
   git clone https://github.com/Stone-People-Like/Web.git .
   ```
   *如果是国内服务器连接 GitHub 慢，可以尝试使用 Gitee 镜像或先下载 zip 包上传（见方式二）。*

### 方式二：在线下载/上传压缩包
1. 点击面板左侧的 **文件**。
2. 进入 `/www/wwwroot/你的域名` 目录。
3. 全选并删除目录下原有的 `index.html` 和 `404.html`。
4. 点击 **远程下载**，URL 填入：
   `https://github.com/Stone-People-Like/Web/archive/refs/heads/main.zip`
5. 下载完成后，你会看到一个 `main.zip`，点击右侧的 **解压**。
6. 解压后可能会多出一层文件夹 `Web-main`。进入该文件夹，**全选**所有文件，点击右上角的 **剪切**。
7. 回到上一级目录（站点根目录），点击 **粘贴**。
8. 确保 `index.html` 文件直接位于站点根目录下。

## 4. 访问与测试
打开浏览器，输入你的域名或 IP，现在应该能看到“常用软件分享”网站了。

## 5. 常见问题与优化

### 开启 HTTPS (SSL) - 去除“不安全”提示
1. 在 **网站** 菜单中，点击你的网站名（或右侧的“设置”按钮）。
2. 在弹出的窗口左侧菜单选择 **SSL**。
3. **如果你有域名**：
   - 选择 **Let's Encrypt** 标签。
   - 勾选你的域名，点击 **申请** 按钮。
   - 等待申请成功后，右上角会出现开关，开启 **强制 HTTPS**。
   - 此时浏览器地址栏的小锁头就会出现，"不安全"字样消失。
4. **如果你使用的是 IP 地址访问**：
   - **注意**：免费的 Let's Encrypt 证书**不支持**纯 IP 地址。
   - 你会一直看到“不安全”提示，这是正常的。
   - 解决方案：去购买一个便宜的域名（如 .top, .xyz 等），解析到你的服务器 IP，然后按照步骤 3 操作。

### 网站更新
如果你修改了代码并 push 到了 GitHub：
- **如果是 Git 部署**：在宝塔终端输入：
  ```bash
  cd /www/wwwroot/你的域名
  git pull
  ```
- **如果是上传部署**：需要重新下载压缩包覆盖。
